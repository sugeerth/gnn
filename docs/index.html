Why This Matters

  • Raw features show overlapping, noisy clusters — the bag-of-words vectors don't capture relational information.
  • GNN embeddings form tight, well-separated clusters because they aggregate neighborhood information through message passing.
  • The NMI improvement from 0.13 → 0.59 shows GNNs extract 4.5x more community signal than features alone.
